# Загрузка накладных на рабочие места начальников отделений почтовой связи

**Название скрипта:** gazette.

**Назначение скрипта:** формирование нового dbf-файла на основе имеющегося, содержащего определённые накладные на печатные СМИ, с изменением некоторых записей в нём и отправка этого файла на FTP-сервер для загрузки на СОУ.

Скрипт используется инженерами группы технической поддержки в случае возникновения сбойных ситуаций с загрузной накладных на печатные СМИ.


Скрипт написан на **Python 2.7**.


Используются модули, не входящие в стандартную поставку интерпретатора:
        - dbfpy.
